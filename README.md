## Start the backend 
# 1. Install Express
npm install express

# 2. Install WebSocket
npm install ws

# 3. (Optional) Install CORS if your frontend will call the backend
npm install cors

# 4. Run the server
node server.js


## Start the Frontend 
npm install
npm run dev
